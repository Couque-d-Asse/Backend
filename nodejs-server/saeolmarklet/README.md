# 🏛️ 새올전자민원창구 자동작성 시스템

PWA + Bookmarklet 기반의 새올전자민원창구 민원 자동 작성 도우미

## 📋 주요 기능

### 1. PWA (Progressive Web App)
- ✅ **사용자 정보 관리**: 이름, 생년월일, 연락처 등 저장
- 📝 **템플릿 관리**: 자주 사용하는 민원 내용 템플릿화
- 🔖 **북마클릿 생성**: 원클릭 자동 입력 북마클릿 생성
- 📜 **작성 이력**: 민원 작성 이력 관리
- 💾 **오프라인 지원**: Service Worker를 통한 오프라인 작동
- 📱 **모바일 최적화**: 반응형 디자인 및 터치 인터페이스

### 2. Bookmarklet
- 🚀 **자동 폼 입력**: 민원 작성 양식 자동 입력
- 🔐 **PASS 인증 지원**: 본인인증 과정 부분 자동화
- ⚡ **원클릭 실행**: 북마크에서 바로 실행
- 📊 **실시간 피드백**: 진행 상황 표시

## 🚀 설치 및 실행

### 1. 의존성 설치
```bash
cd bookmarklet/saeolmarklet
npm install
```

### 2. 환경 변수 설정
```bash
# .env 파일 생성
echo "PORT=8080" > .env
```

### 3. 서버 실행
```bash
# 프로덕션 모드
npm start

# 개발 모드
npm run dev

# PWA만 서빙 (간단한 테스트)
npm run serve
```

### 4. 접속
```
http://localhost:8080
```

## 📱 PWA 설치

1. Chrome/Edge 브라우저로 접속
2. 주소창 우측 설치 아이콘 클릭
3. "설치" 버튼 클릭
4. 홈 화면에서 앱 실행

## 🔖 북마클릿 사용법

### 생성 방법
1. PWA에서 사용자 정보 입력
2. 민원 템플릿 작성
3. "북마클릿" 탭에서 템플릿 선택
4. "북마클릿 생성" 클릭
5. 생성된 링크를 북마크에 추가

### 사용 방법
1. 새올전자민원창구 사이트 접속
2. 로그인 및 민원 작성 페이지 이동
3. 북마크에서 저장된 북마클릿 클릭
4. 자동으로 폼이 입력됨
5. 주소 등 추가 정보 수동 입력
6. 제출

## 🔧 기술 스택

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **PWA**: Service Worker, Web App Manifest
- **Storage**: IndexedDB, LocalStorage
- **Server**: Node.js, Express
- **Security**: Helmet, CORS

## ⚠️ 제한사항

### 자동화 불가 항목
- ❌ **CAPTCHA**: 보안 문자 수동 입력 필요
- ❌ **SMS 인증번호**: 실제 휴대폰 수신 필요
- ❌ **주소 검색**: 팝업창 수동 조작 필요
- ❌ **파일 첨부**: 수동 업로드 필요

### 부분 자동화
- ⚡ **PASS 인증**: 개인정보까지 자동 입력
- ⚡ **민원 양식**: 제목, 내용, 연락처 자동 입력

## 🔒 보안 고려사항

1. **개인정보 보호**
   - 모든 데이터는 로컬 저장
   - 암호화된 IndexedDB 사용
   - 서버 전송 시 HTTPS 필수

2. **북마클릿 보안**
   - 도메인 체크로 피싱 방지
   - XSS 방지 필터링
   - 민감 정보 노출 방지

## 📝 사용 시 주의사항

1. **법적 책임**: 본인의 민원만 작성
2. **정확성**: 자동 입력된 내용 확인 필수
3. **개인정보**: 타인의 정보 입력 금지
4. **서비스 약관**: 새올전자민원창구 이용약관 준수

## 🛠️ 문제 해결

### PWA가 설치되지 않을 때
- HTTPS 환경에서만 설치 가능
- Chrome/Edge 최신 버전 사용
- 시크릿 모드에서는 설치 불가

### 북마클릿이 작동하지 않을 때
- 새올전자민원창구 페이지인지 확인
- 콘솔에서 에러 메시지 확인
- 브라우저 보안 설정 확인

### 자동 입력이 되지 않을 때
- 페이지 구조 변경 확인
- 템플릿 데이터 확인
- LocalStorage 용량 확인
